What's this?

This is an unofficial tool to calculate your CGPA for IITM BS programme students. This is created by Sayan Ghosh, and is not affiliated with IITM in any way. The application is updated as of writing this disclaimer to the current curriculum, however no guarantee is given that it is free of errors.

The source code for this application is available on GitHub. Please create an issue if you find any bugs or have any suggestions.

How CGPA is Calculated?

CGPA or cumulative grade point average is calculated by taking the weighted average of the grade points obtained in all the courses taken by the student. The weightage of each course is the number of credits assigned to it.

How to Use?

The application is fairly simple to use. You can start by clicking on the Start button on the home page. You will be asked to select your current level. After that you will be asked to enter your grades in the courses you have completed. You can leave the grade as '-' if you have not completed the course. After entering the grades you will be shown your CGPA.

Data Collection Policy

This application uses cookies to store your session id. This is used to identify you as a user and to store your data. The data is stored in the server and is not shared with anyone. The data is deleted after 365 days of inactivity. You can delete your data by clicking on the Delete Data button.
