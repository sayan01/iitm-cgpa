What's this?

This is an unofficial tool to calculate your CGPA for IITM BS programme students. This is created by Sayan Ghosh, and is not affiliated with IITM in any way. The application is updated as of writing this disclaimer to the current curriculum, however no guarantee is given that it is free of errors.

The source code for this application is available on GitHub. Please create an issue if you find any bugs or have any suggestions.

How CGPA is Calculated?

CGPA or cumulative grade point average is calculated by taking the weighted average of the grade points obtained in all the courses taken by the student. The weightage of each course is the number of credits assigned to it.
