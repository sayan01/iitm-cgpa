About

This is a simple application to calculate your CGPA for IITM BS Students. It is made by Sayan Ghosh, and is not affiliated with IITM in any way. The application is updated as of writing this disclaimer to the current curriculum, however no guarantee is given that it is free of errors.
The source code for this application is available on GitHub. Please create an issue if you find any bugs or have any suggestions.
